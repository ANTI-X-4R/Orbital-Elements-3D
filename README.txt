🌍 Orbital Elements Visualizer

An interactive 3D visualization of the six classical orbital elements using Three.js.

The application provides a real-time representation of how each orbital parameter changes the geometry and orientation of a satellite's orbit around Earth, making orbital mechanics easier to understand through interactive visualization.

🌐 Live Demo

https://orbital-elements-3d.vercel.app/

---

✨ Features

- 🌎 Interactive 3D Earth visualization
- 🛰️ Real-time satellite orbit simulation
- 🎛️ Interactive control panel
- 📏 Adjustable classical orbital elements:
  - Semi-major Axis (a)
  - Eccentricity (e)
  - Inclination (i)
  - Longitude of Ascending Node (Ω)
  - Argument of Perigee (ω)
  - True Anomaly (ν)
- 🚀 Start / Stop orbital motion
- ↔️ Prograde & Retrograde motion
- 🔄 Reset simulation
- 📊 Dynamic orbital information panel
- ⚠️ Automatic detection of special orbital cases
- 🎨 Modern glassmorphism interface
- ⚡ Smooth real-time rendering

📚 Orbital Elements

Symbol| Description
a| Semi-major Axis
e| Eccentricity
i| Inclination
Ω| Longitude of Ascending Node
ω| Argument of Perigee
ν| True Anomaly

---

🌌 Orbit Classification

Eccentricity| Orbit Type
e = 0| Circular Orbit
0 < e < 1| Elliptical Orbit
e = 1| Parabolic Trajectory
e > 1| Hyperbolic Trajectory
